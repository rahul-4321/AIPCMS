# AI-Powered Content Moderation System (AIPCMS)

AIPCMS is a real-time content moderation API built with FastAPI, Hugging Face Transformers, and PostgreSQL. It provides automated text and image moderation to detect toxicity, hate speech, and inappropriate visuals.

## Features
- **Real-time Text Moderation**: Detects toxicity, threats, insults, and profanity.
- **Image Content Control**: Identifies NSFW content using deep learning models.
- **Audit Logging**: Saves all moderation requests and scores to a PostgreSQL database for review.
- **Scalable Architecture**: Built with FastAPI for high performance.
